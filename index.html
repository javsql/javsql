運営ポリシー

成人向け領域だからこそ、見せ方と扱う情報に明確な線引きを設けています。

✓

18歳未満の閲覧を防止

年齢確認を入口に設置。地域の法令・プラットフォーム規則に合わせて運用します。

✓

APIデータを正しく表示

作品名・価格・画像・販売状態などは、許可されたAPIレスポンスと表示ルールに従い、取得日時を管理します。

✓

公式情報を尊重

DMM.comと提携・所属しているように誤認させず、広告・アフィリエイトであることを明示します。

✓

露骨な素材を転載しない

APIで許可された素材のみを表示し、無断保存・再配布・過度な加工を行いません。

✓

キャッシュと削除依頼に対応

API規約に沿った保存期間を設定し、販売停止・権利者からの連絡に応じて掲載を見直します。

✓

審査前の注意

本ページはUIデモです。実運用にはDMMのアフィリエイト登録、APIキー、個別審査、最新規約の確認が必要です。
